# 8. Riscos e Mitigação

| Risco | Impacto | Mitigação |
|-------|----------|------------|
| Falta de organização do grupo | Atraso na entrega | Divisão clara de tarefas |
| Erros no deploy no Vercel | Documentação fora do ar | Testar build local (`npm run docs:build`) |
| Perda de arquivos | Projeto inviável | Usar GitHub desde o início |
| Erros de diagramação | Nota reduzida | Revisar com modelo do professor |

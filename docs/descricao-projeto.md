# 2. Descrição do Projeto

## 2.1 Visão Geral

O Sistema Coliseum Gym é uma plataforma de gestão para academias que centraliza informações de alunos, planos, pagamentos, treinos e aulas coletivas.  
Seu objetivo é **automatizar processos administrativos**, melhorar o acompanhamento dos treinos e oferecer uma experiência organizada e eficiente para alunos e funcionários.

O sistema permitirá:

- Gerenciamento completo de alunos  
- Controle de pagamentos e planos  
- Registro e acompanhamento de treinos  
- Sistema de reservas de aulas coletivas  
- Geração de relatórios administrativos

## 2.2 Stakeholders

| Stakeholder | Descrição |
|------------|-----------|
| **Aluno** | Utiliza o sistema para consultar treinos, reservar aulas e acompanhar pagamentos. |
| **Instrutor** | Registra treinos, acompanha evolução dos alunos e auxilia nas aulas coletivas. |
| **Administrador** | Gerencia alunos, instrutores, planos, pagamentos e relatórios. |
| **Sistema/Porteiro** | Realiza controle de entrada/saída e valida status financeiro do aluno. |
| **Direção da Academia** | Utiliza relatórios para tomada de decisão. |
